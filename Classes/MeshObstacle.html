MeshObstacle

Extension of SKNode needed to access expanded pathfinding functionality. Three primary purposes are:

  • Create obstacles with points using SKRegion, which has helpers like containsPoint
  • Add real, edge based physics bodies to the world from the CGPath of the SKRegion
  • Create a GKObstacle for a GKGraph to use

Instead of adding sprites through the scene editor and generating physics bodies off of their alpha masks, this class will help to achieve the same result programmatically.

Additionally, expanded functionality could be added down the road, like an obstacle with spikes or different collision masks.

  • region

    Outlines the path of the obstacle. Using this for potential future benefits, but could get away with just storing CGPath. One of the two is needed for containsPoint. Neither would be needed if SKPhysicsBody gained containsPoint. As of iOS9 it doesn’t.
