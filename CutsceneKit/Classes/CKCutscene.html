CKCutscene

public class CKCutscene

Handles cleanup, management and playback of a collection of CKSequence objects that make up a cutscene.

Warning

Because this class deals with closures and callbacks, you must keep the instance of it in scope / reference while it is running.

Starting the Cutscene
  • begin(completion:)

    Begin playing through each CKSequence until completion. Once a CKSequence has been completed, it will be permanently removed from the CKCutscene.

    Note

    There is no way to pause or stop playback. You can control this by only adding CKSequences that you wish to execute now, and delay adding others.
