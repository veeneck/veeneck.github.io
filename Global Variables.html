Global Variables

The following global variables are available globally.

  • scenes

    A list of common scenes to prevent code duplicating. Each individual scene still choose what scene object gets displayed next

  • globalTextureCache

    Global texture cache to minimize loading screens. Cleared only during major scene group changes
