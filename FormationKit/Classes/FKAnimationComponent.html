FKAnimationComponent

class FKAnimationComponent : GKComponent

Handles running a requested animation. Other components will request starting and stopping animations. In theory, this could be expanded to prevent certain state changes (i.e.: cant go from walking to attacking), but that shouldn’t be required for this simulation.

Update Loop
  • updateWithDeltaTime(_:)

    Constantly checking to see if an animation is requested. If so, put it in motion. The animation will run as a SKAction once or on a loop. Requesting .Idle will stop any animation.
