Documentation

Docs are located at veeneck.github.io/FormationKit and are generated with Jazzy. The config file for the documentation is in the project as config.yml. Docs can be generated by running this command in the project directory:

jazzy –config {PATH_TO_REPOSITORY}/FormationKit/FormationKit/config.yml

Note: The output in the config.yml is hard coded to one computer. Edit the file and update the output flag to a valid location.
