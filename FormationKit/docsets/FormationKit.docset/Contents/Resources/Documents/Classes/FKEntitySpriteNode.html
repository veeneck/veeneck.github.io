FKEntitySpriteNode

class FKEntitySpriteNode : SKSpriteNode

Spritekit physcis contacts only contain informaiton about the node in contact, and not the entity. Because of this, we need this empty class to store the node / entity relationship. Then during a physcis event, the entity can be looked up, and the proper component can be called.
