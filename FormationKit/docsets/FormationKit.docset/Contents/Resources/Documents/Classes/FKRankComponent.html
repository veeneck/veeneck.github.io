FKRankComponent

internal class FKRankComponent : GKComponent

This component is necessary for anyone in a squad to properly function. Individuals won’t need it. Used for a handle on squad, standing positions, leader, and potentially more order specific logic.

  • squad

    The squad that this unit belongs to, if any

    Declaration

    Swift

    var squad : FKSquadEntity
  • standingPosition

    Position within the squads formation. Other components may update this. Goal for Unit is to always be where ordered unless a state (i.e.: Combat) has othe rinstructions.

    Swift
