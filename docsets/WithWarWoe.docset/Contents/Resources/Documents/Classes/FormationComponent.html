  • calculateUnitPositions(_:rotation:translate:offset:random:)

    This will calculate the expected position for every unit, given the parameters. Returns an array of those positions.

    • parameter position: The world position to start the calculations from
    • parameter rotation: The heading that the formation should face, in radians
    • parameter translate: If the rotation should translate to a specific unit, defaults to center
    • parameter offset: After rotation, offsets to apply to the units

    Declaration

    Swift
