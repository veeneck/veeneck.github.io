UIAnchorPoint

Used by UINode to mark spots around the edge of the screen that various objects can snap to when they are no longer in view. Main example is the Heraldry component.

  • point

    The center position of the anchor point.

    Declaration

    Swift

    let point : CGPoint
  • occupied

    Bool to indicate if an object is currently using this anchor point, or if it is free.

    Declaration

    Swift

    var occupied : Bool
  • side

    By providing the side of the screen this point lays on, searching for the closet point is much faster. 1: Left 2: Top 3: Right 4: Bottom

    Declaration

    Swift

    let side : CGRectSide
