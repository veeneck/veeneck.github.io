WavePool

Simple tool to manage waves of spawns (i.e.: TowerDefense) in SpriteKit. Goals are:

  • Read from pList
  • Call delegate to handle the actual spawn
  • Keep track of time and cleanup
  • Allow pausing
  • Allow predconditions through willSpawn delegate
